# notifications Service

Part of the Sturdy Waddle PM microservices architecture.

## Status

🚧 **Planned** — This service will be extracted from the modular monolith in a future phase.

## Responsibility

Notification dispatch (in-app, email, Slack, Teams), preference management, and delivery tracking.

## Tech Stack

- **Runtime:** Node.js (TypeScript)
- **Framework:** Hono
- **Communication:** REST API + Kafka events
